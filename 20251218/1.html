예시3

email: 입력값이 이메일 형식(@ 포함 여부 등)인지 자동으로 체크합니다.

예시4

number: 숫자만 입력 가능하며, 우측에 숫자를 조절하는 화살표(스피너)가 생깁니다.

예시5

checkbox: 여러 개를 동시에 선택할 수 있습니다. (다중 선택)

예시6

radio: 같은 name 그룹 중에서 단 하나만 선택할 수 있습니다.

예시7

file: 내 컴퓨터의 파일을 선택하여 업로드할 수 있습니다.

예시8

date: 날짜를 선택할 수 있는 캘린더(달력) UI가 나타납니다.

예시9

color: 색상 선택기(Color Picker)가 나타납니다.

예시10

range: 슬라이더 바 형태의 숫자 입력 도구입니다.

예시11

submit: 입력한 데이터를 서버로 전송합니다.

예시12

reset: 입력한 모든 내용을 초기화합니다.

예시13

button: 아무 기능이 없으며, 자바스크립트를 연결해 커스텀 기능을 넣을 때 씁니다.

예시14

기본값은 태그 사이에 작성합니다.

예시15

select 태그 (드롭다운 선택 리스트)

여러 개의 선택지 중에서 하나(또는 여러 개)를 고를 수 있는

드롭다운 목록을 만듭니다.

내부에는 반드시 option 태그가 포함되어야 합니다.

주요 속성

  • 1) select: 전체 목록을 감싸는 태그입니다.
  • 2) option: 실제 선택할 수 있는 개별 항목들입니다.
  • 3) value: 서버로 전송될 실제 데이터 값입니다.
  • 4) selected: 페이지 로드 시 기본으로 선택되어 있을 항목을 지정합니다.
  • 5) multiple: 여러 개를 동시에 선택할 수 있게 합니다. (Ctrl/Cmd 키 사용)
