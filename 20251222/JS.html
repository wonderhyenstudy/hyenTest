데이터타입과 연산자

변수
데이터 타입
연산자
String : 문자, "" or '' 로 감싸야 한다
Number : 숫자, 수학 계산이 가능함
Boolen : 참(1) 거짓(0), true flase
산술연산자.
+ : 더하기
- : 빼기(별표 사용)
* : 곱하기
/ : 나누기
% : 나머지(나눗셈 후 남은 값. 매우 중요)
문자열 연결 연산자 + 기호 사용함
문자열 String
문자열 + 문자열 = 문자열
문자열 + 숫자 = 문자열

조건문

상황에 따라 실행
비교연산자
> : 크다
< : 적더
>= : 크거나 같다(이상)
<= : 작거나 같아(이하)
=== : 같다(데이터타입, 값도 같아야 같다)
!== : 같지 않다
if 기본 문법
if (조건){ //조건이 참 true 일때만 실행 } else if { //조건이 거짓 false 일때만 실행 }
